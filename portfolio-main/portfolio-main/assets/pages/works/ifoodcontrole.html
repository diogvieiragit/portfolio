Ifood Controle - 2024

Projeto educativo com proposta feita pela universidade FIAP junto ao Ifood. O sistema funciona como uma solução backoffice para os restaurantes cadastrados na plataforma. Sendo possível gerar relatórios específicos, ter acesso ao saldo, pedidos e uma grande variedade de estatísticas.

Toda a identidade visual foi pensada para seguir os padrões da empresa, mantendo fidelidade das cores e dos aspectos visuais em geral. Por se tratar de um projeto educativo, foi utilizado apenas Figma, HTML, CSS, JavaScript e Bootstrap para chegar no resultado final.
